Create `app/core/injury_impact.py` with:

1. Constants:
   ```python
   TIER_IMPACT = {
       "starting_g": 4.0,   # Display only -- not applied (Phase 4 handles)
       "top6_f": 2.0,
       "top4_d": 1.5,
       "bottom6_f": 0.75,
       "bottom_d": 0.75,
   }
   GTD_MULTIPLIER = 0.5
   MAX_INJURY_ADJ = 8.0  # pp cap per team
   FORWARD_POSITIONS = {"C", "L", "R", "LW", "RW", "F"}
   DEFENSE_POSITIONS = {"D"}
   ```

2. `@dataclass(frozen=True) class InjuredPlayer` with fields: team, player_name, position, status, tier, impact_pp. Follows project frozen dataclass pattern.

3. `classify_player_tier(player_name: str, position: str, team_players: list[dict]) -> str`:
   - Match player by last name within team roster (same pattern as Phase 4 goalie resolver)
   - Forwards: rank by toi_per_game among team forwards, top 6 = "top6_f", rest = "bottom6_f"
   - Defense: rank by toi_per_game among team defensemen, top 4 = "top4_d", rest = "bottom_d"
   - Goalies: return "starting_g"
   - Fallback if player not found in roster: use position to assign bottom tier

4. `build_player_tiers(injuries: list[dict], fetch_stats_fn=None) -> dict[tuple[str, str], str]`:
   - Groups injuries by team
   - Fetches player stats only for teams with injuries (lazy, avoids 32 API calls)
   - Uses `fetch_team_player_stats` as default for fetch_stats_fn (injectable for testing)
   - Returns mapping of (team_code, player_name) -> tier string
   - Caches results per team within a single call (no repeated fetches for same team)

5. `calculate_injury_adjustment(home_team: str, away_team: str, injuries: list[dict], player_tiers: dict[tuple[str, str], str]) -> tuple[float, list[InjuredPlayer]]`:
   - For each team, sum per-player impacts (TIER_IMPACT[tier])
   - SKIP goalie injuries in adjustment (tier == "starting_g") -- Phase 4 handles goalie impact via backup save%
   - GTD (status contains "Day-To-Day" or "DTD"): multiply impact by GTD_MULTIPLIER
   - Cap each team's total penalty at MAX_INJURY_ADJ
   - Net = (away_penalty - home_penalty) / 100.0 (convert pp to probability)
   - Also return list of InjuredPlayer objects for dashboard display (including goalies for display)
   - Return tuple: (net_adjustment_float, injured_players_list)
