  <action>
    Add to app/core/multi_season.py (extending the module from Plan 01):

    1. COVID_SEASON = 2020 (the 2020-21 season, 56 games, hub cities)

    2. validate_multi_season(
         seasons: dict[int, list[dict]] | None = None,
         config: TrackerConfig | None = None,
         mode: str = "fixed",  # "fixed" or "grid_search"
         start_season: int = 2015,
         end_season: int = 2024,
       ) -> dict[str, Any]:
       - If seasons dict not provided, call load_seasons(start_season, end_season)
       - Create a single EloTracker instance
       - For each season in chronological order:
         a. If not the first season, call elo_tracker.regress_to_mean()
         b. Run backtest_season(games, config, elo_tracker=elo_tracker) -- Elo carries over
         c. Run evaluate_predictions() on the predictions
         d. Run simulate_betting_roi() on the predictions
         e. Store per-season results: {season, brier_score, accuracy, roi_pct, win_rate, n_predictions, is_covid: season == COVID_SEASON}
         f. If mode == "grid_search": also run grid_search() with a REDUCED grid (3x3x3x3 = 81 combos to avoid runtime explosion):
            half_life: [21, 30, 45], regression_k: [15, 20, 25], home_advantage: [0.10, 0.14, 0.20], logistic_k: [0.7, 0.9, 1.2]
            Store the best params for this season
       - After all seasons: compute overall_pass = all seasons have accuracy >= 0.55 AND roi_pct > 0.0
       - If mode == "grid_search": run analyze_parameter_stability() and determine_verdict()
       - Return structured dict with: season_results, overall_pass, param_stability (if grid_search), verdict (if grid_search), mode, config_used

    3. analyze_parameter_stability(per_season_optimal: dict[int, dict[str, float]]) -> dict[str, dict]:
       For each param (half_life, regression_k, home_advantage, logistic_k):
       - Compute mean, stdev, min, max, coefficient_of_variation (stdev/mean)
       - Store values_by_season mapping
       - Use statistics.mean and statistics.stdev (or pstdev for small N)

    4. determine_verdict(season_results: list[dict], param_stability: dict | None = None) -> str:
       - Check 1: all_pass = every season has accuracy >= 0.55 and roi_pct > 0.0
       - Check 2: high_drift = any param has CV > 0.3 (if param_stability provided)
       - Return one of:
         "VERDICT: Parameters are STABLE across seasons" (all_pass and not high_drift)
         "VERDICT: Model performs well but parameters show DRIFT -- current params may be season-specific" (all_pass and high_drift)
         "VERDICT: Parameters are OVERFIT -- model fails on held-out seasons" (not all_pass)

    Add corresponding tests to tests/core/test_multi_season.py using synthetic data. Mock backtest_season, evaluate_predictions, simulate_betting_roi, and grid_search to return controlled results for testing the orchestration logic without actual computation. This keeps tests fast.
